To run it:

Ensure that MongoDB accepts connection to `mongodb://localhost/todo`.

```
  git clone
  npm install
  npm start
  open http://localhost:3000
```

The todo app supports:

- Login/SignUp/Logout
- Add todo
- Edit todo (click on table fields)
- Remove todo (hover on todo and click on the right icon)
